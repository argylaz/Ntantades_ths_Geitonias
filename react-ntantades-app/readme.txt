 Η ιστοσελίδα έχει υλοποιηθεί ακολουθώντας όσο πιο πιστά μπορούσαμε τα wireframes της 2ης εργασίας. Πέρα από κάποιες μικρές αλλαγές που έγιναν (κυρίως οπτικές), η αρχική δομή των σελίδων έχει παραμείνει περίπου ίδια. Η υλοποίηση έγινε με την χρήση react, MUI και firebase (μόνο για database, χωρις  για την βάση δεδομένων. Με την χρήση αυτών καταφέραμε να υπάρχει σχεδόν όλη η επιθυμητή λειτουργικότητα (εγγραφές, αλλαγές στις δομές του database, breadcrumb κλπ) επιδιώκοντας παράλληλα τη μέγιστη χρηστικότητα.
